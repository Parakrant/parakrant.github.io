Objective:
  • The objective of this study is to developed story TTS systems in Hindi using syllables as basic unit.

  • We have developed two story TTS systems.
    • The first system is developed using neutral TTS system and second is a unit selection based TTS system developed using story speech corpus. The first system uses the neutral TTS system to convert the neutral speech into story style speech. The conversion is carried out by our three proposed modules. These modules can be easily plugged into the neutral TTS system. The speech generated by the neutral TTS system is modified into story style using the three modules. The three modules designed are:
      1. Story-specific emotion detection (SSED) module
      2. Story-specific prosody generation (SSPG) module
      3. Story-specific prosody incorporation (SSPI) module.


    • The Figure given displays synthesis framework of the first story TTS system. The Story-specific prosody generation (SSPG) module include prosody rules which are derived for five story-specific emotions (i.e., happy, fear, sad, anger, and neutral) by analyzing the difference in prosodic parameters of neutral and original storyteller speech utterances. The neutral TTS system with these designed modules can be able to generate story style speech.
    • The second TTS system is a corpus based synthesis system. It uses unit section to appropriately select the units (i.e., syllable) and concatenate them (using signal processing techniques) to generate story style speech. The efficiency of the story TTS system depends on how good the linguistic, semantic and the prosodic information are extracted from the text, and how well these informations are incorporated into the synthesized speech. In corpus based synthesis, the story TTS system is build from the story corpus recorded by a professional human storyteller. Here, the objective is to enhance the synthesis quality of synthesized story style speech. To enhance the quality, we need to improve the prosody models of story TTS system. In view of this, prosody modeling frameworks are proposed for duration, intonation, intensity and pause pattern of story style speech.

  • The scope of the work is described are as follows:
    1. Development of Hindi story TTS system using the neutral TTS system.
    2. Development of Hindi story TTS system using a story corpus.
    3. Modeling of story-specific pauses for story style speech synthesis
    4. Prosody (i.e. duration, intonation, and intensity) modeling of the syllables based on story genre.
